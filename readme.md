# 八、列表
## 无序列表
语法：无序列表用 - + * 任何一种都可以

- 列表内容
+ 列表内容
* 列表内容

## 有序列表
语法：数字加点  
1.列表内容   
2.列表内容  
3.列表内容  

## 列表嵌套
上一级和下一级之间敲三个空格即可  

- 一级列表  
   - 二级列表  
   + 二级列表


# 九、表格
语法：   
表头|表头|表头   
---|:--:|---:   
内容|内容|内容   
内容|内容|内容   

第二行分割表头和内容。   
- 有一个就行，为了对齐，多加了几个   
文字默认居左
- 两边加：表示文字居中
- 右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略
